## 🌐 About

A **multi-page business landing website** built with pure HTML & CSS — no frameworks, no JavaScript, no dependencies. Covers the complete structure of a real-world company site: hero section, navigation, services page, about page, and a contact page — all linked and styled from a single stylesheet.

> Built during my **CodSoft Frontend Internship** as a hands-on exercise in converting a design layout into a clean, navigable multi-page HTML website.

---

## 📄 Pages

| File | Page | What's On It |
|:-----|:-----|:-------------|
| `index.html` | 🏠 **Home** | Header nav · Hero section ("Your One-Stop Solution") · CTA button · Footer |
| `about.html` | 👤 **About** | Company/brand background and intro |
| `services.html` | 🛠️ **Services** | Service listings and offerings |
| `contact.html` | 📬 **Contact** | Contact details or form |
| `styles.css` | 🎨 **Stylesheet** | Global styles shared across all 4 pages |

---

## 🗂️ Project Structure

```
landing-page-/
├── index.html      # Home — hero, nav, footer
├── about.html      # About page
├── services.html   # Services page
├── contact.html    # Contact page
└── styles.css      # Shared stylesheet for all pages
```

---

## ✅ Features

- **4-page site** — fully linked navigation across Home → About → Services → Contact
- **Hero section** — headline, subtext, and a call-to-action button
- **Semantic HTML5** — `<header>`, `<nav>`, `<section>`, `<footer>` used throughout
- **Single shared CSS** — one `styles.css` drives the look of all pages consistently
- **Zero dependencies** — opens instantly in any browser, no setup needed
- **Mobile-aware** — viewport meta tag included for responsive behaviour
